# Новые функции мессенджера

## ✅ Редактирование и удаление сообщений
- **БД**: Добавлены колонки `edited_at` и `deleted` в таблицу messages
- **API**:
  - `PUT /api/messages/:messageId` - редактирование сообщения
  - `DELETE /api/messages/:messageId` - удаление сообщения (мягкое удаление)
- **UI**: Контекстное меню на правой кнопке мыши (ПКМ) и долгом нажатии (мобильные устройства)
  - Доступны эмодзи реакции прямо в меню
  - Кнопки редактирования и удаления для своих сообщений

## 🔍 Поиск по чатам и сообщениям
- **БД**: Используется `LIKE` оператор для гибкого поиска
- **API**: `GET /api/search?q=query` - поиск чатов по имени и сообщений по тексту
- **UI**: Строка поиска в сайдбаре (фильтрует чаты во время ввода)

## ✔️ Статусы сообщений
- **Использует существующую колонку `status`** в таблице messages
- **Три статуса**: 
  - `sent` ✓ - отправлено
  - `delivered` ✓✓ - доставлено
  - `read` ✓✓ - прочитано
- **UI**: Галочки Telegram-style (✓/✓✓) отображаются после времени в сообщении
- Статус `read` устанавливается при загрузке чата

## 🖼️ Аватары пользователей
- **БД**: Поле `avatar` в таблице users (уже было, но улучшено)
- **API**: `POST /api/user/avatar` - загрузка аватара пользователя (multer подключен)
- **UI**: Аватары отображаются:
  - В сайдбаре в списке чатов
  - В шапке текущего чата
  - Рядом с каждым сообщением
  - Если нет изображения, показывается первая буква имени

## 😊 Реакции на сообщения
- **БД**: Новая таблица `reactions` (message_id, user_id, emoji)
- **API**:
  - `POST /api/reactions` - добавление реакции
  - `DELETE /api/reactions/:messageId/:emoji` - удаление реакции
- **UI**: Панель с 5 популярными эмодзи (👍 ❤️ 😂 😢 🔥) в контекстном меню
- Реакции отображаются под сообщением в виде пузырьков

## ↩️ Ответы на сообщения
- **БД**: Добавлена колонка `reply_to_id` в таблицу messages
- **Функция**: Готова для будущей реализации (структура создана)
- Позволит цитировать сообщения и показывать квоты в UI

## 🔔 Уведомления браузера
- **API**: Использует Notification API браузера
- **Функции**:
  - `requestNotificationPermission()` - запрашивает разрешение при входе
  - `showNotification()` - показывает уведомление при новом сообщении в неактивной вкладке
  - Уведомления показываются только когда вкладка неактивна (`document.hidden`)

## 👤 Профиль пользователя
- **API**: 
  - `GET /api/user` - получение данных профиля
  - `POST /api/change-password` - изменение пароля
- **UI**: Кнопки в модалке настроек:
  - **Профиль** - показывает имя, код, email, дату регистрации
  - **Сменить пароль** - интерактивное изменение пароля (3 подсказки)

## 🎨 Улучшения UI/UX
- **Стили для новых компонентов**:
  - Поле поиска в сайдбаре
  - Контекстное меню сообщений
  - Реакции с эффектами при наведении
  - Статусные иконки сообщений
  - Индикатор "отредактировано" на сообщении
  - Темная тема (автоматически применяется ко всем новым элементам)

## 📱 Мобильная поддержка
- Контекстное меню работает с долгим нажатием (touchstart/touchend)
- Все кнопки и элементы оптимизированы для сенсорного экрана
- Реакции легко добавлять свайпом вверх

## 🔧 Технические детали

### Новые таблицы БД:
```sql
-- Реакции
CREATE TABLE reactions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    message_id INTEGER NOT NULL,
    user_id INTEGER NOT NULL,
    emoji TEXT NOT NULL,
    UNIQUE(message_id, user_id, emoji),
    FOREIGN KEY (message_id) REFERENCES messages(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### Новые колонки:
```sql
-- В таблице messages:
ALTER TABLE messages ADD COLUMN edited_at TEXT;
ALTER TABLE messages ADD COLUMN deleted INTEGER DEFAULT 0;
ALTER TABLE messages ADD COLUMN reply_to_id INTEGER;
```

## 🚀 Как использовать новые функции

### Редактирование сообщения:
1. Правой кнопкой мыши по своему сообщению
2. Нажать "✏️ Редактировать"
3. Ввести новый текст в диалоге

### Добавление реакции:
1. Правой кнопкой мыши по сообщению
2. Выбрать эмодзи из панели реакций

### Поиск:
1. Ввести текст в поле "Поиск..." в сайдбаре
2. Увидеть отфильтрованные чаты в реальном времени

### Уведомления:
1. При первом входе браузер запросит разрешение на уведомления
2. Нажать "Разрешить"
3. Получать уведомления о новых сообщениях при неактивной вкладке

### Смена пароля:
1. Нажать кнопку "⚙️ Настройки"
2. Нажать "Сменить пароль"
3. Ввести текущий пароль и новый пароль (2 раза)
